Here are some useful array methods:

  • indexOf() ‚Äì Finds the position of a specific element in the array (returns -1 if not found).
  • includes() ‚Äì Checks if a certain value exists in the array (returns true or false).
  • pop() ‚Äì Removes the last element from the array and returns it.
  • push() ‚Äì Adds one or more elements to the end of the array and returns the new array length.
  • shift() ‚Äì Removes the first element from the array and returns it.
  • sort() ‚Äì Arranges the elements in the array in ascending order (by default as strings).
  • splice() ‚Äì Adds, removes, or replaces elements in the array at specific positions.
  • reverse() ‚Äì Reverses the order of elements in the array in-place.
