Built-in Functions

JavaScript comes with a bunch of built-in functions that do a lot of the useful things for you. These functions are ready to use without requiring any imports, or installs. If only Java could be so generous. Here are a few of the notable functions:

  • charAt(index) – Grabs the character at a specific index in a string.
  • indexOf('char') – Finds the first appearance of a character (or substring) in a string.
  • charCodeAt(index) – Gives you the ASCII value of the character at the specified position.
  • fromCharCode(code) – Converts an ASCII number back into a character.
  • replace('old', 'new') – Substitutes one substring with another.
  • split('delimiter') – Breaks a string into an array based on a separator.
  • toUpperCase() – Converts every letter in the string to uppercase (yelling, basically).
  • toLowerCase() – Brings everything down to lowercase (library voice).
  • join('glue') – Takes array pieces and stitches them back into a single string.
  • Math.pow(base, exponent) – Raises a base number to the power of an exponent.
  • Math.min() – Finds the smallest number among the arguments.
  • Math.max() – Finds the largest number among the arguments.
  • Math.round() – Rounds a number to the nearest whole number (goodbye decimals).

Some Code in Action

// Let's work with a string value, of course we have got to keep it iconic. let greeting = "Hello World";
console.log(greeting.charAt(6)); // W
console.log(greeting.indexOf("o")); // 4
console.log(greeting.replace("World", "Sir")); // Hello Sir | now does that not sound like Jarvis?

Output:

W 4 Hello Sir
// Now let's mess with some numbers.
console.log(Math.pow(2, 5)); // 32
console.log(Math.min(5, -10, 42)); // -10
console.log(Math.round(3.6)); // 4

Output:

32 -10 4
// Finally, upper and lower case transformations.
let name = "Jarvis";
console.log(name.toUpperCase()); // JARVIS
console.log(name.toLowerCase()); // jarvis

Output:

JARVIS jarvis
