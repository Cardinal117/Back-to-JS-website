Conditional Statements:

Conditionals, the fun parts of programming. The if else is a very useful statement that is used probably everywhere. Games are basically built using this important statement.

if else

To jog your memory I have some if else code in action below and because we are in a similar topic I have also added some switch case code below that:

// Remember our variable from before? No? // Well here they are again.
let name = "REDACTED"; let funnyNumber = 69; // Hehe, Nice let enjoysPizza = true; /* Will only be true if both name equals REDACTED and number is bigger or equals to 69 and of course skip everything and pass if enjoysPizza is true. */ if (name === "REDACTED" && funnyNumber >= 69 || enjoysPizza){
console.log("Ahh welcome pizza enjoyer"); // Prints this if true. } else // If conditions aren't met the next condition is checked. if (name === "Master-Chief" || number === 117) { // Welcome Master-Chief. console.log("ATTENTION!"); } else { // And of course, if no conditions are met default to the else statement. console.log("Non pizza enjoyers will be prosecuted!"); }
// Let's check the name variable but with switch case. switch (name) { case "REDACTED": console.log("Ahh welcome pizza enjoyer"); break;
case "Master-Chief": console.log("ATTENTION!"); break;
default: console.log("Non pizza enjoyers will be prosecuted!"); break; }
