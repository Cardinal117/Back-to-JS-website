DOM Events

Events allow JavaScript to react to interactions on the webpage. Here are some common ones:

  • onchange – Triggered when a form element is modified.
  • onclick – Triggered when an element is clicked.
  • onmouseover – Triggered when hovering over an element.
  • onmouseout – Triggered when the cursor leaves an element.
  • onkeydown – Triggered when a keyboard key is pressed.
  • onload – Triggered when the HTML document has finished loading.
