jQuery

jQuery simplifies DOM manipulation using shorter syntax and built-in utilities(yes, making it way easier to do tedious tasks). Here is something interesting, see that menu top left? Well I made that menu using jQuery and I think that is pretty cool. Here’s a quick jQuery cheat sheet/reminder list:

Basic Structure:

$(function() { // jQuery code goes here });

Common Selectors:

  • $("h1") – Selects all <h1> elements.
  • $("#mainHeading") – Selects element with ID mainHeading.
  • $(".yellow-font") – Selects all elements with class yellow.
  • $("*") – Selects all HTML elements.
  • $(this) – Refers to the current selected element.
  • $("button") – Selects all <button> elements.
  • $("[href]") – Selects elements with an href attribute.

Creating & Appending Elements:

// This adds the text below in between the the <p></p> elements. let newParagraph = $("<p></p>").text("Salutations fellow programmer!");
$("body").append(newParagraph);// This simply appends the <p></p> element above to the body of the page.

Common jQuery Functions:

  • after() – Adds content after the selected element.
  • append() – Inserts content at the end of the selected element.
  • before() – Adds content before the selected element.
  • empty() – Clears the content but keeps the element.
  • prepend() – Inserts content at the start of the selected element.
  • text() – Gets or sets text of the selected element.
  • val() – Gets or sets form field value.
  • remove() – Removes the element entirely.

Linking External jQuery Scripts:

<head>   <script src= "https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>   <script src= "script.js"></script> </head>

Events & Effects:

// Select all paragraph elements $("p").click(function() { // Add a click event handler   // Hide the clicked paragraph slowly   $(this).hide('slow'); });

Common jQuery Events:

  • Mouse Events: click, dblclick, mouseenter
  • Keyboard Events: keypress, keydown, keyup
  • Form Events: submit, change, blur

Chaining Effects:

// Select all button elements $("button") // Change text color to blue .css("color", "blue") // Slide the button down over 500ms .slideDown(500) // Then slide it back up over 500ms .slideUp(500);

Common jQuery Effects:

  • hide() – Hides elements
  • show() – Displays elements
  • fadeIn(), fadeOut() – Fades elements in or out
  • slideUp(), slideDown() – Slides content
  • animate() – Animates properties
  • stop() – Stops ongoing effects
