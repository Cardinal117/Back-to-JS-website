User Defined Functions

While built-in functions are quite powerful, JavaScript allows you make your own and this is where the real fun happens. User defined functions allow you to wrap up reusable blocks of code into a neat easily accessible package. They make your code cleaner, cooler, and reusable like a pro.

Examples in Action

// This is how you declare a function: function greet(name) { // Notice how you can add parameters to functions.
console.log("Hello, " + name + "!"); }
// This is how you call a Function: greet("Tony Stark"); // Output: Hello, Tony Stark!

But Be Warned! Not all functions should be allowed t o touch the grace of a loop.

While it might be tempting to put any of your hand crafted perfection of a function inside a loop, be careful. Some functions take a ton of performance(Like database calls, animations, or sound effects surprisingly). Placing them in a loop without being cautious will haunt you and your computer:

  • Performance slowdowns are highly likely
  • Infinite noise or error spam sounds will drive you crazy
  • Browser crashes or CPU overheat (yes, even in 2025 that is still a thing)

Why Use User-Defined Functions?

  • They allow you to reuse code allowing for less redundancy and less bugs.
  • They help organize logic into bite sized chunks.
  • They make your code look cleaner and more professional.(unless you haphazardly chuck them in loops)
