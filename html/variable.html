Variables:

The important thing to understand about variables is that they can vary. As you might of remembered in Javascript there are 4 different types of variables:

  • var - This is the original way to declare variables however, this way is prone to hoisting issues and is mostly avoided in modern js.
  • let â€“ This is the way you are familiar with, and is used when the variable's value will change over time.
  • const - The way to declare constants, this is used to declare a variable where the value cannot be changed.
  • Global Variables - This is normally declared outside any function, is accessible anywhere however should be used sparingly to avoid namespace conflicts and annoying bugs.

Variable.Common_Data_Types

As you know, Javascript supports multiple core data types. Here are the most commonly used ones to jog your memory:

  • String - A bunch or characters, like the iconic "hello world!". A string can contain just about any characters, numbers and of course letters.
    • let name = "REDACTED"; // You can also use '' for string values. let name = 'YEET';
  • Number Variables - These include integers, doubles, and floats.
    • // integer let funnyNumber = 69; // Hehe, Nice // double/float let iconicNumber = 399.99;
  • Boolean - Represents true or false values.
    • let enjoysPizza = true;
  • Array - An ordered list of values.
    • let colors = ["red", "green", "blue"];
